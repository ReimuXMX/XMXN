  Run commad in your shell:

  via curl

  #+BEGIN_SRC
  curl -L https://github.com/re-mx/mxnanorc/raw/master/install.sh | sh
  #+END_SRC

  via wget

  #+BEGIN_SRC
  wget --no-check-certificate https://github.com/re-mx/mxnanorc/raw/master/install.sh -O - | sh
  #+END_SRC
